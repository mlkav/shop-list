# Submission Akhir Aplikasi Android Sederhana

Aplikasi ini menampilkan daftar **15 online shop populer** baik lokal maupun global. Pengguna dapat melihat detail setiap toko, termasuk kategori, deskripsi lengkap, pendiri, tahun berdiri, lokasi, dan mengunjungi website resmi toko tersebut.
